Comet Chat
==========

* Ruby 1.8.7+ or 1.9.2+
* sinatra-cometio with Sinatra1.3+


Install Dependencies
--------------------

    % gem install bundler
    % bundle install


Run
---

    % rackup config.ru -p 5000

=> http://localhost:5000


CUI Client
----------

    % ruby bin/cui_chat_client.rb
